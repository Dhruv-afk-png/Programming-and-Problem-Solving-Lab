# Problem 01:-
Q1. Write a Python program that calculates the area of a circle when the radius is provided by the user. Use π = 3.14 and display the area.

# Algorithm:-

Step 1: Start.
Step 2: Input the radius r.
Step 3: Set π = 3.14.
Step 4: Calculate area = π × r × r.
Step 5: Display the area.
Step 6: Stop.
